# Bellas XV Marketing Splash

Single-page marketing site at shopbellasxv.com.
Static HTML/CSS, no build step.

Served by nginx from this directory directly.

## Assets

- `assets/wordmark.svg` - BXV wordmark used in hero and footer
- `assets/logo.svg` - standalone logo used as the favicon
- `assets/hero-source.png` - original hero photograph source, if available
- `assets/hero-desktop.webp` - hero photograph, desktop crop
- `assets/hero-desktop.jpg` - hero photograph, desktop JPEG fallback
- `assets/hero-mobile.webp` - hero photograph, mobile vertical crop
- `assets/hero-mobile.jpg` - hero photograph, mobile JPEG fallback
- `assets/booking-placeholder.webp` - secondary photograph for booking section
- `assets/booking-placeholder.jpg` - secondary photograph JPEG fallback
- `assets/og-image.jpg` - social share image

## Editing

Edit `index.html` and `styles.css` directly. No build, no deploy script.
Changes go live as soon as files are saved and nginx config is in place.
